## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm ci
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Postman endpoints

click [this link](https://www.getpostman.com/collections/d70a6d35e048591ee0e6) to get postman json file. Now you can simply import it in your postman & send your requests
